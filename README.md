# 공공 API

소프트웨어와 웹 개발에 사용하기 위한 무료 API 모음

이 프로젝트에 대한 컨트리뷰트 정보는 [컨트리뷰트 가이드](.github/CONTRIBUTING.md)를 확인해주세요.

이 프로젝트는 한국어 사용 개발자를 위해 [public-apis](https://github.com/public-apis/public-apis) 저장소를 포크하여 번역한 저장소입니다.
